## 简介

本 Skill 是国家卫生健康委发布的**《成人高尿酸血症与痛风食养指南（2024年版）》**的完整蒸馏版本，将官方指南转化为 AI 可直接使用的结构化知识体系。

## 使用方法

1. 加载本 Skill 后，AI 将自动获得完整的指南知识库
2. 用户可用自然语言询问任何高尿酸/痛风相关问题
3. AI 根据 system_prompt.md 的7模块框架进行通俗化应答
4. 涉及食谱、嘌呤查询时自动调用对应数据文件

## 快速开始

**用户问**："我尿酸500，能吃什么？"

**AI 应答流程**：
1. 判断属于高尿酸血症（>420μmol/L切点）
2. 从 KPK-10 提供优选/不宜食物清单
3. 从 KPK-09 精确匹配常见食物嘌呤值
4. 从 KPK-06 初步判断可能证型并给出建议
5. 推荐对应阶段食谱（KPK-12 无症状期食谱）
6. 给出实操方案和常见误区提醒

## 中医证型与食养方索引

### 湿浊证（无症状/间歇期）
- 茶饮：橘皮茶 / 砂仁橘皮茶 / 菊苣薏苡仁茶
- 药膳：茯苓橘皮粥 / 山药茯苓粥 / 山药橘皮小米粥

### 湿热证（急性发作期）
- 茶饮：金银花荷叶茶 / 玉米须白茅根茶
- 药膳：冬瓜薏苡仁豆腐汤 / 马齿苋金银花薏苡仁粥 / 马齿苋炒鸡蛋 / 海带冬瓜薏苡仁汤 / 菊苣蔬菜饼

### 痰瘀证（间歇期/慢性期）
- 茶饮：橘皮莱菔子茶 / 桔梗橘皮茶
- 药膳：桃仁橘皮粥 / 当归桔梗煮鸡蛋 / 冬瓜薏苡仁山楂粥

### 脾肾亏虚证（慢性期）
- 茶饮：山药茯苓茶 / 人参黄芪茶
- 药膳：山药核桃仁粥 / 山药黄精粥 / 山药木耳炒莴笋
