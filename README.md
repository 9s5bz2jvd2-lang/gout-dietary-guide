## 指南出处

- **全称**：《成人高尿酸血症与痛风食养指南（2024年版）》
- **发布单位**：国家卫生健康委办公厅

## 功能特点

- **嘌呤数据全覆盖**：400+种食物精确mg/100g嘌呤值，红绿灯分级
- **分期膳食指导**：无症状期 vs 急性发作期，不同阶段不同策略
- **食养原则**：8条官方原则，核心围绕"控嘌呤、限酒精、多饮水"
- **中医食养**：4种证型（湿浊/湿热/痰瘀/脾肾亏虚）+ 24个食养方（9茶饮+15药膳）
- **地区食谱**：3大地区（内陆/沿海/高原）×4季节×2阶段×3能量级 = 72套
- **食物交换表**：7类食物等量交换，灵活配餐
- **科普风格**：说人话、给具象量、指误区——精准但不端着

## 关键数据速查

| 项目 | 推荐量 | 通俗理解 |
|------|--------|---------|
| 每日饮水 | ≥2000mL | 约8杯水，促进尿酸排泄 |
| 嘌呤摄入（急性期） | <150mg/天 | 比一个鸡蛋的嘌呤还少 |
| 嘌呤摄入（缓解期） | <300mg/天 | 控制总量，不是不吃 |
| 酒精 | 急性期禁酒 | 啤酒最危险，白酒其次 |
| 高果糖饮料 | 不喝或少喝 | 果糖代谢直接产尿酸 |
| 食盐 | <5g/天 | 一个啤酒瓶盖 |
| 烹调油 | ≤25g/天 | 两汤匙封顶 |
| 血尿酸目标 | <360μmol/L（无痛风石）/ <300μmol/L（有痛风石） | 达标是关键 |
